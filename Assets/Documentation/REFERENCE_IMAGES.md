# Reference Images

Os PNGs desta pasta e de `Assets/Resources/ReferenceImages` sao alvos fiduciais ficticios gerados para teste academico. Eles nao usam arte oficial nem conteudo protegido.

Para testar:

1. Imprima os PNGs em tamanho aproximado de carta/marker.
2. Evite brilho e reflexos.
3. Use boa iluminacao.
4. Rode `Magic AR Assistant > Preparar projeto para build` para marcar as texturas como legiveis.
5. Abra `ARMatchScene` ou inicie pelo menu.
